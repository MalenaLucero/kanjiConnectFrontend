About

Hello. My name is Malena, I'm a front-end developer and I've been studying Japanese since 2016. I created this application to review Japanese vocabulary because I was never happy with the spaced repetition apps I was using. I wanted to have more control over the data and the way it was presented for review. Here I'll give a small tour of the application to explain how it works. For doubts or queries, please contact kanjiconnect@gmail.com

Repositories and database

Front-end

Angular application with Angular Material. I'm currently using Rxjs to manage the state and I try to keep it updated with the lastest Angular release.

Back-end

Nest.js application that I run locally for the auth, POST, UPDATE and DELETE endpoints. It used to be hosted on Heroku.

Next.js application currently hosted on Vercel for the public GET enpoints.

Database

Mongo Atlas

Entities

Expressions

Words or phrases that are uploaded to be stored and reviewed. Expressions are uploaded through https://kanji-connect.vercel.app/study/upload/expression and uploaded expressions can be filtered through https://kanji-connect.vercel.app/study/manage/expressions.

When an expression is written in the input field of the Upload section, the Autocomplete button gets the reading, English meaning, JLPT level and transitivity from Jisho.org.

Expressions can also be associated with a Lesson, a Source and given an example sentence. Expressions can also have Tags.

Once an expression is uploaded, the kanji of that expression are mapped and those kanji are stored as user-kanji.

User-kanji

User-kanji are kanji uploaded indirectly by the user. In the front-end, they appear as 'kanji' and can be filtered though https://kanji-connect.vercel.app/study/manage/user-kanji.

When a user uploads an expression, the kanji of that expression are stored as user-kanji, meaning kanji that they 'belong' to that user. In this way, when the user looks up a specific kanji in the application, they'll also get all the words that they've uploaded with that kanji.

Lesson and Souce

A Lesson is an entity made up of Sources. Every example sentence has a Source. Lessons are a way to group Expressions based on when they were studied for the first time. I have classes once every two weeks, so for me Lessons group everything that I do during that time. The Sources are the specific materials I use, for example, a chapter of a book, an anime episode, a podcast.

Review

Once an expression is uploaded it can be reviewed through https://kanji-connect.vercel.app/study/review. 'Show a hint' shows the example sentence. If the user presses Hard, the difficulty of the expression increases. If the user clicks on Easy, the difficulty decreases.

Both expressions and kanji can be reviewed.
